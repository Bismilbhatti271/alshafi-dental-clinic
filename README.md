# 🦷 Alshafi Dental Clinic

Premium dental clinic website built with Next.js, Tailwind CSS, and Font Awesome.

## Pages

- **Home** — Hero with SVG graphics, stats, services, testimonials carousel, gallery
- **About** — Clinic story, values, timeline, team
- **Services** — 12 dental services with detailed cards
- **Gallery** — Filterable clinic image gallery
- **FAQ** — Accordion-style frequently asked questions
- **Contact** — Booking form + WhatsApp direct booking

## Tech Stack

- [Next.js 16](https://nextjs.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Font Awesome 6](https://fontawesome.com/)
- Google Fonts (Bricolage Grotesque, DM Sans, Instrument Serif)

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
pnpm build
```
